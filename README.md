# ROVOX
🎮 特徴

ボクセル上の世界を舞台にしたゲーム

銛を指して抜き、敵や障害物を砕くアクション

コントローラー操作対応

🕹 操作方法（コントローラー）

左スティック：移動

右スティック：視点移動

R1：銛を投げる / 回収

R2：引き抜きモードに移行

A：銛を引っ張る

🔧 使用技術

C#/Mathematica/ShaderLab

⏱ 制作期間

約6か月

🌐 実行方法

Unity バージョン 2022.3.21f1 で開き実行可能

📌 制作意図・工夫

全てをボクセルで作ると処理が追いつかないため、ボクセル状に見せかける表現を採用

狙いにくい部分に補正ロックを付けて操作性を向上

UIを工夫して、操作や状況が直感的に分かるように設計
